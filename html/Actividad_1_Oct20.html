Importancia de Semantica

Es importante usar los elementos apropiados:

  • Para organizar y crear código limpio y fácil de leer.
  • Para que los usuarios con necesidades especiales tengan acceso a plataformas que utilizan elementos específicos para facilitar el uso de la página.
  • Para optimizaciones de búsqueda en navegadores (Google, DuckDuckGo, etc.).

div

Un contenedor general.

header

Donde se da un abstract/introduccion a la pagina.

main

El contenido queda encapsulado por este tag.

nav

Contiene la barra de navegacion.

article

Un ejemplo es en un blog, cada post se contiene dentro de un article tag.

section

Uso general es para separar secciones del contenido.

aside

Mueve el elemento a un lado de otro elemento. Normalmente se usa para ads o para agregar contexto al elemento vecino si se ocupa.

footer

Se usa al final del documento para demostrar licensias/items legales/nombre del sitio

Definiciones de Tags

h1-h6

Son headings que se usan para dar importancia a los detalles que siguen. Entre mas bajo el numero mas importancia tiene. (Font-size mas grande y bold)

p

Un parafo que se usa para texto general. Normalmente sigue despues de un heading.

br

Un break que crea un espacio entre los elementos verticalmente.

hr

Un break que crea un espacio entre los elementos horizontalmente.

blockquote

Es un bloque que contiene un quote.

pre

Mantiene el formato de un texto como esta representado en el HTML.

cite

Se usa para referir a un libro/articulo/etc..

em

Cambia las letras a italicos.

q

Version inline de blockquote.

b

Cambia las letras a bold.

i

Cambia las letras a italicos.

strong

Cambia las letras a bold.

ul

Empieza un contenedor para una lista desordenada. (Puntos)

ol

Empieza un contenedor para una lista ordenada. (Numerado)

li

Un item para una lista.

u

Subraya las letras.

mark

Resalta el texto

small

Cambia font-size a un tamaño pequeño

sub

Crea un subindice (_)

sup

Crea un superindice (^)

abbr

Agrega un acronimo con un contexto de algun titulo.

code

Se usa para mostrar un snippet de codigo.

del

Raya el texto con una linea por el medio.
